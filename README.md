# FloraVision 🌿

A responsive plant-themed frontend website built using **Next.js**, **React**, **TypeScript**, and **Tailwind CSS**. The project features a modern UI with reusable components, smooth animations, and a fully responsive design.

## Tech Stack

- Next.js
- React.js
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React Icons

## Installation

```bash
npm install
npm run dev
```

Open **http://localhost:3000** in your browser.
